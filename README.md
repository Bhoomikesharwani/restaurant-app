# 🍽️ Restaurant App

A modern React-based restaurant web application that allows users to browse the menu, order food online, and reserve tables. The project includes a React frontend and a Node.js/Express backend, providing a responsive and user-friendly dining experience.

## ✨ Features

- 🍕 Browse food menu
- 🛒 Order food online
- 📅 Reserve tables
- 📱 Responsive design
- ⚡ Modern React UI
- 🔗 Backend API integration
-     fast 2 sec Ai response

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB

## 🚀 Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```
